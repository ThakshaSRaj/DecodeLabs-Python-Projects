# DecodeLabs Python Projects

## 📌 Overview

This repository contains three Python projects completed as part of the **DecodeLabs Python Internship**. These projects demonstrate the fundamentals of Python programming, including user input, loops, functions, conditionals, and the use of built-in libraries.

## 🚀 Projects

### 1. Expense Tracker

**Description:**
A simple application that allows users to enter expenses and calculates the total amount spent.

**Features:**

* Accepts multiple expense inputs
* Calculates the total expense
* Easy-to-use command-line interface

---

### 2. To-Do List

**Description:**
A basic task management application that helps users organize their daily tasks.

**Features:**

* Add new tasks
* View existing tasks
* Simple and user-friendly interface

---

### 3. Random Password Generator

**Description:**
Generates strong and random passwords using letters, numbers, and special characters.

**Features:**

* User-defined password length
* Random and secure password generation
* Uses Python's built-in libraries

## 🛠️ Technologies Used

* Python 3

## ▶️ How to Run

1. Clone or download this repository.
2. Open a terminal in the project folder.
3. Run any project using:

```bash
python expense_tracker.py
python todo.py
python password_generator.py
```
